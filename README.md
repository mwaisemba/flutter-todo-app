## Features
- Add new tasks with a title
- View the list of tasks in a scrollable format
- Mark tasks as complete (with checkbox and strike-through)
- Delete tasks (swipe left or delete button)
- Persist tasks locally using `shared_preferences` (data remains after closing and reopening the app)
- Pre-loaded with 5 sample tasks on first run (for test demonstration)
